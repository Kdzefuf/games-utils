SkyPilot — Aviation Companion App
SkyPilot — это мобильное приложение для любителей авиации, будущих пилотов и энтузиастов полётов. Приложение сочетает образовательные, симуляционные и практические функции в одном удобном интерфейсе.

Разработано на React Native CLI
Поддержка Android
Полностью работоспособный MVP с 5 разделами

Основные функции
Приложение включает 5 полноценных разделов:

Главная — краткое описание и навигация
Flight Simulator — интерактивный симулятор полёта:
Управление высотой, скоростью и креном
Визуализация земли и неба
Предупреждение о сваливании (STALL) при низкой скорости
Aircraft Database — каталог реальных самолётов с изображениями и характеристиками
Weather Radar — информация о погоде в ключевых аэропортах Индии
Pilot Logbook — личный журнал полётов с сохранением данных через AsyncStorage
Все экраны адаптированы под разные размеры экранов и протестированы на Android.

Технологии
React Native CLI
TypeScript
React Navigation (Bottom Tabs)

Установка и запуск
Требования
Node.js ≥ 18
Android SDK
Java JDK 17

Шаги
Установите зависимости:
```
npm install
```
Запустите Metro:
```
npm start
```
Соберите и запустите на устройстве или эмуляторе:
```
npm run sndroid
```
Структура проекта

Сборка для публикации
Для генерации APK:

```
cd android
./gradlew assembleRelease
```
Файл будет доступен по пути:
android/app/build/outputs/apk/release/app-release.apk
